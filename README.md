# Parallaxy

Create beautiful images parallax with attributes or in javascript

## Update

[See the changelog](./CHANGELOG.md)

## Installation

```
$ npm i parallaxy-img
```

Or with the cdn

```html
<script src="https://unpkg.com/parallaxy-img@2.0.0/dist/parallaxy.js"></script>
```
